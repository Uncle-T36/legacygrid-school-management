Configuration Instructions

To update your Stripe configuration:

  1. Log in to your Stripe Dashboard
  2. Navigate to Developers → API keys
  3. Copy your test mode public and secret keys
  4. Update the following settings in your settings.py file:
STRIPE_PUBLIC_KEY = 'pk_test_your_actual_public_key_here'
STRIPE_SECRET_KEY = 'sk_test_your_actual_secret_key_here'
STRIPE_PRICE_ID = 'price_your_actual_price_id_here'
